# ⚡ БЫСТРАЯ ЗАГРУЗКА НА GITHUB

## 🚀 Выполните эти команды по порядку:

### 1. Установите Git (если не установлен)
- Скачайте с https://git-scm.com/download/win
- Установите и перезапустите PowerShell

### 2. Настройте Git
```bash
git config --global user.name "Ваше Имя"
git config --global user.email "ваш@email.com"
```

### 3. Инициализируйте репозиторий
```bash
git init
git add .
git commit -m "Initial commit: Facade Orders app"
```

### 4. Создайте репозиторий на GitHub
- В открывшемся браузере создайте репозиторий `facade-orders`
- Сделайте его **ПУБЛИЧНЫМ**
- **НЕ** добавляйте README, .gitignore, лицензию

### 5. Подключите к GitHub
```bash
git remote add origin https://github.com/YOUR_USERNAME/facade-orders.git
git branch -M main
git push -u origin main
```

**Замените `YOUR_USERNAME` на ваш GitHub username!**

## ✅ Готово!

Ваш проект будет доступен по адресу:
`https://github.com/YOUR_USERNAME/facade-orders`
