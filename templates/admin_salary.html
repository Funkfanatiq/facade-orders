Отчет по зарплатам

Информация

Для просмотра детального отчета по зарплатам выберите месяц и год, затем нажмите "Сформировать отчет".

Отчет будет содержать:

  • Общее количество часов по каждому сотруднику
  • Расчет зарплаты по часовой ставке
  • Детализацию по дням
  • Суммарную информацию по всем сотрудникам
