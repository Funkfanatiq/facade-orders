# 🔑 БЫСТРАЯ ШПАРГАЛКА ДЛЯ ВХОДА

## 👥 ВСЕ РАБОЧИЕ МЕСТА И ПАРОЛИ

| Роль | Логин | Пароль | Функции |
|------|-------|--------|---------|
| **Админ** | `admin` | `admin123` | Все функции |
| **Менеджер** | `manager` | `manager123` | Управление заказами |
| **Производство** | `prod` | `prod123` | Работа с заказами |
| **Фрезеровка** | `frez` | `frez123` | Фрезерные работы |
| **Шлифовка** | `shlif` | `shlif123` | Шлифовальные работы |
| **Монитор** | `monitor` | `monitor123` | Просмотр заказов |

## 🚀 Быстрый вход для тестирования:

### 1. **Админ** - полное тестирование
- Логин: `admin`
- Пароль: `admin123`
- **Доступ:** все функции системы

### 2. **Менеджер** - управление заказами
- Логин: `manager`
- Пароль: `manager123`
- **Доступ:** создание и управление заказами

### 3. **Производство** - работа с заказами
- Логин: `prod`
- Пароль: `prod123`
- **Доступ:** изменение статуса заказов

### 4. **Фрезеровка** - фрезерные работы
- Логин: `frez`
- Пароль: `frez123`
- **Доступ:** работа с фрезерными заказами

### 5. **Шлифовка** - шлифовальные работы
- Логин: `shlif`
- Пароль: `shlif123`
- **Доступ:** работа с шлифовальными заказами

### 6. **Монитор** - просмотр заказов
- Логин: `monitor`
- Пароль: `monitor123`
- **Доступ:** только просмотр

## 📋 Для развертывания на Render:

После инициализации базы данных все эти аккаунты будут созданы автоматически.

## 🔧 Если нужно изменить пароли:

1. Отредактируйте файлы:
   - `app.py` (команда init-db)
   - `init_db.py`
   - `render_init_db.py`

2. Пересоздайте базу данных на Render

---

**💡 Совет:** Начните тестирование с роли **Админ** для полного доступа ко всем функциям!
